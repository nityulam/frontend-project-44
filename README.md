<h3>Description</h3>
<hr>
<p>Mind Games is a set of five console games based on the principle of popular mobile brainâ€”boosting applications. Each game asks questions that need to be answered correctly. After three correct answers, it is considered that the game is over. Incorrect answers end the game and offer to go through it again.</p>

<h3>Installing the game:</h3>
<p>$ make install</p>
<p>$ npm link</p>

To start the game, you will need:
Node.js - v20.15.1
